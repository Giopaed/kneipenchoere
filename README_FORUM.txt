FORUM-UPDATE – Kneipenchöre vernetzt
=======================================

Diese ZIP ergänzt ein echtes moderiertes Forum auf der Website.

Enthaltene Dateien:
- forum.html          → neue Forum-Seite
- forum.js            → lädt und filtert Forum-Beiträge
- Kneipenchor.css     → ergänzt das Design für das Forum
- daten/forum.json    → veröffentlichte Forum-Beiträge
- admin/config.yml    → CMS wird um den Bereich „Forum“ erweitert

So hochladen:
1. ZIP entpacken.
2. GitHub öffnen.
3. Repository Giopaed/kneipenchoere öffnen.
4. Add file → Upload files.
5. Alle Dateien/Ordner aus diesem entpackten Paket hochladen.
6. Commit message: Forum entwickelt
7. Commit changes klicken.
8. 1–2 Minuten warten.
9. https://kneipenchor.net/forum.html öffnen.

Wie das Forum funktioniert:
- Besucher können unten auf der Forum-Seite einen Beitrag vorschlagen.
- Der Beitrag landet in Netlify Forms und wird NICHT automatisch veröffentlicht.
- Die Redaktion prüft den Beitrag.
- Veröffentlichte Beiträge werden im CMS unter /admin/ → Forum → Forum-Beiträge eingetragen.
- Danach erscheinen sie auf der Forum-Seite.

Wichtig:
- Keine privaten Telefonnummern, Passwörter oder internen Daten öffentlich eintragen.
- Öffentliche E-Mail-Adressen nur veröffentlichen, wenn die Person zugestimmt hat.
